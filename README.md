# PE04 – Vehicle, Car, and Drivable (Java)

## Description
This program demonstrates basic object-oriented programming concepts in Java, including inheritance, interfaces, and method overriding. A Car class extends a base Vehicle class and implements the Drivable interface.

## Program Features
- Uses inheritance (Car extends Vehicle)
- Uses an interface (Drivable)
- Overrides methods to display detailed information
- Adds year and price attributes to the Car class
- Runs from a main application class to test functionality

## Classes
- Vehicle – Base class that stores the vehicle brand
- Car – Subclass of Vehicle that adds model, year, and price
- Drivable – Interface defining start and stop behavior
- VehicleApp – Main class that runs the program

## Example Output
Brand: Honda
Model: Civic
Year: 2026
Price: $40000.0
Civic is starting.
Civic is stopping.

## How to Run
Run the VehicleApp class. The program will create a Car object, display its details, and simulate starting and stopping the vehicle.
